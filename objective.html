Memory Madness!

Test your memory with increasingly complicated memory challenges.

Memory Madness has fifteen levels. Each level consists of three games. Each game requires you to remember the pattern in which dots are displayed to you and then click on the same dots, in the same order.

Level one is three games of one dot each. Level two is three games of two dots, and so on. As you climb through the levels, the challenge to your memory increases. Beware! If you click on the wrong dot, you are bumped back to the previousl level.
